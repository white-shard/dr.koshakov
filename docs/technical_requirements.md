Заголовки:

- h2: Размер шрифта: 32px, цвет черный
- h3: Размер шрифта: 20px, цвет #374151

Текст: Размер шрифта: 16px, цвет #374151

Иконки: Lucide

Скругление: 8px

Отступы:
- 64px между блоками
- 20px между карточками

Страницы:
- Главная (Лендинг)
